Marcado de áreas

Para marcar un área primero se la debe seleccionar pintándola con el ratón, o con una combinación de teclado y ratón al mantener apretada la tecla shift después de seleccionar la celda inicial y la celda final.

Luego se puede hacer click en los botones "Area", "Titular", "Encabezado", "Descartar" o "Fin"; según se desee.

Para desmarcar un área se puede hacer click derecho sobre alguna celda de la misma para luego hacer click en "Desmarcar Área/Encabezado/etc.".

Para ver las propiedades de un área marcada, se procede igual que el paso anterior pero se selecciona "Opciones área Encabezado/Fin/Descartar".
