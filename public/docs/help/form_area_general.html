Propiedades del área

Nombre (obligatorio): indica el nombre del área de importación para así diferenciarlas de las otras áreas; el nombre indicado debe ser único en todas las áreas de las hojas.

Rango filas (valor numérico): para saber cuantas filas tendrá que moverse verticalmente para encontrar el área en las hojas Excel a importar.

Rango columnas (valor numérico): para saber cuantas columnas tendrá que moverse horizontalmente para encontrar el área en las hojas Excel a importar.

Área fija: indica si el área siempre tiene esa cantidad de filas y columnas. No se puede marcar si se marcó un área fin.

Iterar filas o columnas: indica si el área de importación crece en filas o en columnas.
