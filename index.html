motoEx

MotoEx es una herramienta fácil de usar que permite la importación de datos desde hojas electrónicas.

Problema

Uno de los mayores problemas que encontre durante el desarrollo de várias aplicaciones fue la resistencia que los usuarios tenian al usarlas, debido a que trabajaban con hojas electrónicas "casi siempre Excel", consideraban doble trabajo tener que llenar un formulario en el cual copiaran su información y les doy la razón en este punto, pero el principal problema reside en la cantidad de formatos que puede existir de las misma información. Los usuarios no son ordenados para crear su información, generalmente no existe un formato pre definido en el cual deben trabajar la misma y según sean sus necesidades van creando formatos. En varias ocaciones realize importaciones desde excel tomando en cuenta el formato del usuario, pero los usuarios tambien cambian su formato con el tiempo lo cual hace que el código creado para la importación se vuelva obsoleto y necesite modificaciones y en algunos casos reescribirlo.

Respuesta

Crear un sistema que permita la importación de datos desde hojas electrónicas en las cuales se pueda definir de forma simple los formatos mas comunes que los usuarios usen. MotoEx ayuda a solucionar este problema, es verdad que no solucionará todos los problemas, pero en gran medida es una solución rápida. Esta herramienta fue desarrollada dentro del equipo que desarrallaba la política nacional de hidrocarburos de Bolivia en el año 2009.

Instalación

Para poder usar motoEx es necesario tener instalado Ruby y Ruby on Rails además de php5 y php5-cli para instalar Ruby les recomiendo este vínculo que muestra paso a paso como instalar un stack de Ruby en Ubuntu. Para poder instalar en Ubuntu o debian php5 y php5-cli solo ejecuten el comando 

    sudo apt-get install php5 php5-cli
luego es necesario instalar algunas gemas y rails de la siguiente forma 
    sudo gem install rails less

Ahora hay clonar el proyecto que se encuentra en http://github.com/boriscy/motoEx/ 

    git clone git://github.com/boriscy/motoEx.git
En caso de que no les funcione git por el firewall o no puedan instalar, pueden descargar de: http://github.com/boriscy/motoEx/downloads

Ahora ingresen a la aplicación y ejecuten 

    sudo rake gems:install
    rake db:migrate
    rake db:seed

La primera línea instala todas las gemas necesarias para que la aplicación funcione, la segunda línea crea la base de datos y la última linea llena los datos necesarios para que la aplicación pueda funcionar. Ahora es momento de inciar el servidor ejecutando 

    ruby script/server
Y despues vayan a la dirección http://localhost:3000 e ingresen usuario "admin", contraseña "demo123"
